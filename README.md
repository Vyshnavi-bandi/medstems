# Healthcare Website UI

This project is a responsive healthcare website built using **Next.js, React, Bootstrap, and CSS**. The goal of this project is to create a clean and modern user interface for a medical or hospital-related website.

The website includes common sections used in healthcare platforms such as a navigation bar, hero section, feature highlights, services, appointment form, and footer.

## Technologies Used

* Next.js
* React.js
* Bootstrap 5
* CSS
* Font Awesome (for icons)

## Features

* Responsive navigation bar with dropdown menu
* Hero section with patient statistics card
* Feature highlight section
* Services section
* Center of Excellence section
* Appointment booking form
* Disclaimer section
* Footer with social media icons

## How to Run the Project

1. Install dependencies

```
npm install
```

2. Start the development server

```
npm run dev
```

3. Open the browser and visit

```
http://localhost:3000
```
